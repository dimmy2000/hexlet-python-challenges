## Fibonacci

Реализуйте функцию `fib()`, находящую положительные Числа Фибоначчи.
Аргументом функции является порядковый номер числа.

Формула:
```commandline
f(0) = 0
f(1) = 1
f(n) = f(n-1) + f(n-2)
```

```python
>>> from solution import fib
>>> fib(3)
2
>>> fib(5)
5
>>> fib(10)
55
>>>
```
