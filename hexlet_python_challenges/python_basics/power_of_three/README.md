## Power of three

Реализуйте функцию `is_power_of_three()`, которая определяет, является ли
переданное число натуральной степенью тройки. Например, число 27 — это третья
степень: `3 ** 3`, а 81 — это четвёртая: `3 ** 4`.

Пример:

```python
>>> is_power_of_three(1)
True
>>> is_power_of_three(2)
False
>>> is_power_of_three(9)
True
```
